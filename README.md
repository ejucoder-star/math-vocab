# 📐 国际课程数学英语词汇发音词典

一个为国际课程数学教师和学生设计的英语词汇学习工具，支持**英式发音**即时播放，涵盖 IGCSE、A-Level 和 AP 三大课程体系。

---

## ✨ 功能特色

- 🔊 **一键播放英式发音** — 使用浏览器内置语音引擎，无需网络即可播放
- 📖 **剑桥词典直达** — 每个词汇附带剑桥官网链接，可收听真人录音
- 🔍 **实时搜索** — 支持中文、英文、音标同时搜索
- 📚 **10大专题分类** — 按知识领域快速筛选
- 📱 **响应式设计** — 手机和电脑均可正常使用
- 💾 **单文件** — 整个应用只有一个 HTML 文件，无需安装

## 🛠️ 技术说明

- 纯 HTML + CSS + JavaScript，无任何外部依赖
- 发音功能基于 [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API)
- 优先选择英式英语语音（en-GB），自动降级到系统默认语音
